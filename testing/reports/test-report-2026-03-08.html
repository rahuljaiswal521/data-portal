Fix 1 — BronzePage.test.tsx: Missing SWR mock caused 5 test failures
Root cause
The useTestSuites() hook calls useSWR() internally. Without a per-file mock, isLoading remained true so the component rendered skeleton placeholders — all role/text assertions failed.
